# NAAMA Mentor Connect

A React + TypeScript + Vite web application for the National Arab American Medical Association's mentorship platform.

## Project Structure

- `src/App.tsx` — Main application component (all UI, state, and data)
- `src/index.tsx` — React entry point
- `index.html` — HTML shell
- `vite.config.ts` — Vite configuration (port 5000)
- `tsconfig.json` — TypeScript configuration

## Running the App

```
npm run dev
```

Runs on port 5000.

## Features

- Onboarding flow (role selection, profile setup, focus areas)
- Mentor discovery with search and filters
- Relationships / connections view
- Session scheduling (calendar + time slots)
- Profile page
- Dual-role (mentor + mentee) support
